# Управление проектом через .bat файлы

Всего 3 батника для управления всем проектом:

## 1. START.bat ⭐ (ГЛАВНЫЙ)

**Двойной клик - и всё работает!**

Что делает:
- При первом запуске: установит все зависимости через setup.bat
- При обычном запуске: открывает интерактивное меню

```bash
Просто двойной клик на START.bat
```

---

## 2. menu.bat (Интерактивное меню)

Запускается автоматически через START.bat.

Можно запустить отдельно:
```bash
menu.bat
```

Опции меню:
1. **Import MLS Data** - импорт CSV файла
2. **Update Street Colors** - анализ улиц
3. **Generate Map** - создать HTML карту
4. **Check Email** - проверить Gmail на новые участки
5. **Start Telegram Bot** - запустить бота
6. **Run Full Pipeline** - полный анализ (1+2+3)
7. **Database Stats** - статистика БД
8. **Exit** - выход

---

## 3. setup.bat (Первая установка)

Запускается автоматически при первом START.bat.

Можно запустить вручную:
```bash
setup.bat
```

Что делает:
1. Проверяет Python
2. Создает venv
3. Устанавливает зависимости из requirements.txt
4. Создает .env из шаблона

**После setup.bat обязательно заполнить .env!**

---

## Рабочий процесс:

### Первый раз:
1. Двойной клик `START.bat`
2. Ждем установку
3. Редактируем `.env` файл (DATABASE_URL, TELEGRAM_BOT_TOKEN)
4. Снова двойной клик `START.bat`

### Каждый день:
1. Двойной клик `START.bat`
2. Выбираем опцию в меню

---

## Примеры:

### Импортировать новые данные:
```
START.bat → опция 1 → указать путь к CSV
```

### Сгенерировать карту:
```
START.bat → опция 3
```

### Полный анализ:
```
START.bat → опция 6
```

---

**Все просто: один клик на START.bat → выбираешь опцию → готово!**
