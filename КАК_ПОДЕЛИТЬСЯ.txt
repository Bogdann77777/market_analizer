========================================================
              ШАГ ЗА ШАГОМ (ПЕРВЫЙ РАЗ):
========================================================

ШАГ 1: Скачай ngrok
   → Открой: https://ngrok.com/download
   → Нажми "Download for Windows"
   → Получишь файл ngrok.zip
   → Распакуй ngrok.exe в папку E:\market_analyzer\

ШАГ 2: Запусти SHARE_ONLINE.bat
   → Двойной клик по файлу
   → Подожди 5-10 секунд

ШАГ 3: Найди URL в окне
   → Ищи строку с "Forwarding"
   → Там будет ссылка типа:
     https://1234-56-78-90-123.ngrok-free.app
   → Скопируй её

ШАГ 4: Отправь брату
   → Скинь ему эту ссылку в мессенджер
   → Он открывает → видит твою карту!

========================================================
           ЧТО ДЕЛАТЬ В СЛЕДУЮЩИЙ РАЗ:
========================================================

1. Просто запусти SHARE_ONLINE.bat
2. Скопируй URL
3. Отправь брату

Всё! Ngrok уже будет скачан, ничего повторно делать не нужно.

========================================================
                  ВАЖНЫЕ МОМЕНТЫ:
========================================================

✓ НЕ закрывай окно пока брат пользуется
  (Закроешь окно = ссылка перестанет работать)

✓ Каждый раз НОВЫЙ URL
  (Это нормально, ngrok так работает)

✓ Локально всегда работает: http://localhost:5001
  (Для тебя это быстрее)

✓ Брату может показать предупреждение
  (Просто нажать "Visit Site" или "Continue")

========================================================
              ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ:
========================================================

Проблема: "ngrok.exe not found"
→ Значит не скачал или положил не туда
→ Скачай и положи в E:\market_analyzer\

Проблема: Сервер не запускается
→ Проверь что START_WEB_UI.bat работает сначала
→ Открой http://localhost:5001 - должна быть карта

Проблема: URL не открывается у брата
→ Проверь что окно SHARE_ONLINE.bat ещё открыто
→ Скопируй URL заново
